Class JpegSegmentReader

java.lang.Object
  com.drew.imaging.jpeg.JpegSegmentReader

public class JpegSegmentReader
extends java.lang.Object

Performs read functions of JPEG files, returning specific file segments.

JPEG files are composed of a sequence of consecutive JPEG 'segments'. Each is identified by one of a set of byte values, modelled in the JpegSegmentType enumeration. Use readSegments to read out the some or all segments into a JpegSegmentData object, from which the raw JPEG segment byte arrays may be accessed.
